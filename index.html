Welcome to HTML5 Webpack!

Terminal

Run the following command to clone the repository:

git clone https://github.com/sudo-self/html5.git

Navigate to the Project Directory

Change to the project directory:

cd html5

Install and Start

Install the necessary packages:

npm install

Run the following command to start the application:

npm start

Project is Live

Access the application at the following link:

localhost:8080

Development Server

Start the development server with Webpack:

webpack serve --open --config webpack.config.dev.js
